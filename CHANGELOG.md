## [0.1.0-beta] — 2026-07-11

Перший публічний бета-реліз Focus ON.

### Додано
- **Таймер фокусу:** 4 режими (Deep Work 50хв, Фокус 25хв, Коротка 15хв, Перерва 5хв) + власний час
- **Категорії діяльності:** DeepWork 🧠, Креатив 🎨, Навчання 📚, Читання 📖, Тренування 💪, Інше ✨
- **Музика:** підтримка YouTube, YouTube Music, SoundCloud, Spotify, Apple Music + прямі аудіо-URL
- **Автоотримання назви та виконавця** через oEmbed (YT/SoundCloud/Spotify)
- **Бажане (❤️) та закріплення (📌)** треків
- **Автовідтворення** наступного треку з тієї ж категорії
- **Локальний кеш аудіо** (IndexedDB) — швидкий старт, офлайн
- **Статистика (free):** сьогодні / всього / сесій + за режимами
- **Преміум-статистика:** графіки за періодами, за категоріями, серії (streaks), повна історія
- **Оплата через Telegram Stars** (100⭐/міс) — нативна валюта Telegram
- **Промокоди:** активування користувачем + CRUD для адміна
- **Профіль:** аватар, тариф, мережева інформація (IP, країна, місто, провайдер, швидкість)
- **Рекомендації** (адаптивні під активність)
- **Кеш музики:** розмір, очистити
- **Повідомити про баг** → адміну в Telegram з логами + мережею
- **Автолог крашів:** глобальний error/unhandledrejection handler
- **Адмін-дашборд:** користувачі (DAU/WAU/MAU), сесії, прослуховування, доход, топ-користувачі з лінками
- **Адмін-треки:** додавання в «Демо» та «Для всіх», видалення, перейменування
- **STOP:** довге утримання на таймері = повне скидання
- **Свайп-видалення** треків + потрійний тап для перейменування
- **Світла/темна тема** з перемикачем
- **Десктоп-адаптація** (max-width 480px, центрування)
- **Постійна БД:** Neon PostgreSQL (дані не зникають при деплої)
- **Keepalive:** GitHub Actions + UptimeRobot (Render не засинає)
- **Telegram Webhook:** прийом платежів Stars + /start команди

### Технічне
- Backend: Python FastAPI + PostgreSQL (Neon) + Telegram Bot API
- Frontend: Vanilla JS, Apple-style design, «Режим Бога» неонова тема
- Розгортання: Render (веб) + Neon (БД)
- SQLite/PostgreSQL compatibility layer (автопереклад SQL діалектів)
- HMAC-SHA256 авторизація initData, BIGINT для tg_id

### Відомі обмеження
- Фонове відтворення музики обмежене WebKit (музика грає поки Telegram активний)
- Spotify/Apple Music embed — 30с прев'ю без входу в акаунт
- Neon scale-to-zero: перший запит після простою ~1-2с
